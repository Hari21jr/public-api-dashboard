# Public API Dashboard

A simple dashboard that displays data from public APIs:
- Current weather for a selected city
- Latest tech news headlines
- (Planned) crypto / stock market data

## Tech Stack
- HTML, CSS, JavaScript
- Fetch API for calling public APIs

## Features
- Weather card showing temperature, feels-like, and conditions
- News card with clickable headlines
- Placeholder card for crypto/stocks
