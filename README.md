# Public API Dashboard

A simple web dashboard that displays data from public APIs:
- Current weather for a selected city
- Latest tech news headlines
- (Planned) crypto / stock market data

## Tech Stack

- Python (Flask)
- HTML, CSS
- Fetching JSON data from public APIs

## Features

- Weather card showing temperature, feels-like, and conditions for Ottawa
- News card with clickable headlines from a public news/tech feed
- Placeholder card for future crypto / stock market data
